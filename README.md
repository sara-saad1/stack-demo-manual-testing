Manual Testing Project – Stack Demo Website
This project involved performing Manual Testing on the Stack Demo website to ensure the quality and reliability of its functionalities. The work included:

Requirement analysis to understand the website’s features and expected behavior.

Creating and executing Test Cases to cover key functionalities such as login, product browsing, shopping cart, and order placement.

Bug Reporting to document any issues found, including steps to reproduce, expected vs. actual results, severity, and priority.

Through this project, I was able to identify and document bugs systematically, helping improve the website’s stability and providing a better user experience
